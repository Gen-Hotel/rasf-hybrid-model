# Metabolic Reprogramming in Rheumatoid Arthritis Synovial Fibroblasts: a Hybrid Modeling Approach

## Scope

This repository summarizes all the codes and data used to generate the results presented in [Metabolic Reprogramming in Rheumatoid Arthritis Synovial Fibroblasts: a Hybrid Modeling Approach, Aghakhani et al. (2022)](https://www.makeareadme.com/). Please refer to the latter for all information concerning the methods used.
